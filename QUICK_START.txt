# UniTrack - Simplified Project

## Setup Database

```bash
psql -U postgres -c "CREATE DATABASE unitrack_db;"
psql -U postgres -d unitrack_db -f database/unitrack_schema.sql
```

## Run Python Version (Flask)

```bash
cd python_version
pip install -r requirements.txt
python app.py
```

Access: `http://localhost:5000`

**Login Credentials:**
- Username: `admin`
- Password: `password`

## Run PHP Version

```bash
cd php_version
php -S localhost:8000
```

Access: `http://localhost:8000`

**Login Credentials:**
- Username: `admin`
- Password: `password`

## Features

### Student Management (CRUD)
- **Create** - Add new students with validation
- **Read** - List all students with details
- **Update** - Edit student information
- **Delete** - Remove student records

### Attendance Tracking
- **Mark** - Record attendance for students
- **View** - Filter and display attendance records

### Validation
- Email format validation
- Enrollment number uniqueness
- Required field checks

## Core Functions to Compare (4 Functions)

1. **Authentication** (`login()` in Python, `login.php` in PHP)
2. **Add Student** (`add_student()` in Python, `add_student.php` in PHP)
3. **View Attendance** (`view_attendance()` in Python, `view_attendance.php` in PHP)
4. **Mark Attendance** (`mark_attendance()` in Python, `mark_attendance.php` in PHP)
